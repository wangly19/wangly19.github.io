# CSS

# 选择器优先级别

  • 第一梯队：内置样式
  • 第二梯队： ID选择器
  • 第三梯队： calss选择器
  • 第四梯队： 标签选择器

# link和@import的区别

  • link是老版本标签，是属于XHTML，而@import是css提供的
  • link在页面被加载时会同时被加载，而@import需要等到页面被加载完成在加载。
  • @import只有在IE5上才能兼容，而link能被所有兼容，当然现在已经无所谓了
  • link权重比@import高
  • @import不能被dom控制，所以 js无法去使用dom改变样式

# em/px/rem单位

  • px是绝对单位，按照页面像素精确展示
  • em：相对单位，基准点为父节点字体大小
  • rem相对单位，以根节点字体大小来计算

# 如何理解z-index

z-index（层级）是CSS属性控制重叠元素的加载顺序，默认为0。就好像洋葱一层一层的拨开一样，中心点在中间。

# 盒子模型

浏览器在对文档进行 布局，渲染引擎会根据标准之一的CSS基础盒模型，将所有元素表示为一个个矩形盒子 ，然后 CSS对这个盒子进行装饰。 盒模型由content（内容）、padding（内边距）、border（边框）、margin（外边距）组成。

标准盒模型：元素的宽度 = margin-left + border-left + padding-left + width + padding-right + border-right + margin-right

边框盒模型： 元素占据的宽度 = margin-left + width + margin-right

边距 盒模型：火狐的私有模型，基本没有人用

在开发中一般直接设置标准盒模型

box-sizing: content-box;
