# 栈

栈是一种常见的数据结构，它需要去遵从

  1. 进栈 push(value)
  2. 出栈 pop()
  3. 栈顶 peek()
  4. 栈的长度 size()
  5. 清空栈 clear()
  6. 判断栈是否有数据 isEmpty()
  7. 输出栈 outpuy()
class Stack{

    constructor(){
        this.item = new Array()
    }

    // 入栈方法
    push( element ){
        return this.item.push(element)   
    }

    // 出栈方法
    pop(){
        return this.item.pop()    
    }

    // 返回栈顶
    peek(){
        return this.item[this.item.length-1]
    }

    //检查栈
    isEmpty(){
        return this.item.length === 0 || false    
    }

    // 栈长
    size(){
        return this.item.length
    }

    // 清除栈
    clear(){
        this.item = []
        return this.isEmpty()
    }

    output(){
        return this.item.toString()
    }
}

# 队列

队列和栈的不同在于，栈是先进后出，队列是先进先出，依次排序，和食堂打饭排队一般。包含以下方法

  1. 入队enqueue()
  2. 出队dequeue()
  3. 队首front()
  4. 判断队是否有数据isEmpty()
  5. 队长size()
  6. 清空队clear()
  7. 输出队output()
class Queue{
    constructor(){
        this.item = new Array()
    }

    // 入队
    enqueue(element){
        return this.item.push(element)
    }

    // 出队
    dequeue(){
        return this.item.shift()
    }

    // 队首
    front(){
        return this.item[0]
    }

    // 检查队列
    isEmpty(){
        return this.item.length === 0 || false
    }

    // 队长度
    size(){
        return this.item.length
    }

    // 清除队列
    clear(){
        this.item = []
        return this.isEmpty()
    }

    // 输出队列
    output(){
        return this.item.toString()
    }
}
